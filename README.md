# advent-of-code

My solutions for Advent of Code.
* 2019 - Python
* 2020 - Go
* 2021 - Go
* 2022 - Rust
* 2023 - Go

### Usage
* Download: `go run cmd/download.go -y <year> -d <day> -l <language>`
* Go: `go run cmd/main.go <year> <day> ...`
* Rust: `cargo run --bin=<day>` from the year's directory.
